ROUTE(8) BSD System Manager’s Manual ROUTE(8) NAME route — manually manipulate the routing tables DESCRIPTION Route is a utility used to manually manipulate the network routing tables. It normally is not needed, as a system routing table management daemon such as routed(8), should tend to this task.
